# How to Connect the CSS file to HTML

Easy steps. You gotta make the following changes in the HTML document.

## The .CSS document

```html
<head>
<link rel="stylesheet" href="path/styles.css">
</head>
```
**BONUS**:
#### The Favicon - check below.

```html
<head>
<link rel="icon" href="favicon.ico">
</head>
```
